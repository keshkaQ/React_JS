# Рецепт борща — React-приложение

Простое веб-приложение на React, отображающее кулинарный рецепт борща.

## Технологии

- **React 19** — функциональные компоненты
- **Vite** — сборка и dev-сервер
- **JSX** — синтаксис для описания UI

## Структура компонентов

| Компонент | Описание |
|-----------|----------|
| `Header.jsx` | Заголовок с названием рецепта |
| `MainImage.jsx` | Фотография готового блюда |
| `ProductList.jsx` | Список ингредиентов с количеством |
| `CookingSteps.jsx` | Пошаговая инструкция приготовления |
| `Footer.jsx` | Подвал сайта |

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется по адресу: **http://localhost:5173**

## Сборка

```bash
npm run build
```
