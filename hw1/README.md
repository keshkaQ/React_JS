# Домашнее задание №1

Первое домашнее задание на Node.js. Точка входа: `app.js`.

## Что делает программа

При запуске выводит в консоль (с раскраской через внешний npm-пакет `chalk`):

- приветствие пользователя (функция из `utils/message.js`, подключение своего модуля)
- результат вычисления квадрата числа `3` (функция из `utils/message.js`, подключение своего модуля);
- версию Node.js;
- текущие дату и время (локаль `ru-RU`);
- текущую директорию проекта;
- информацию об ОС (тип, домашняя папка, имя компьютера) через встроенный модуль `os`;
- информацию о файле `app.js` (имя, расширение, папка) через встроенный модуль `path`.


## Требования

- Node.js и npm.

Проект проверен на Node.js `v24.11.1`.

## Установка зависимостей

```bash
npm install
```

## Запуск

```bash
npm start
```

Эквивалентно:

```bash
node app.js
```
